Kitura-Cache

Kitura cache

Summary

Kitura thread-safe in-memory cache.

API

Initialization:

  public init(defaultTTL: UInt = 0, checkFrequency: UInt = 600)

Where: - defaultTTL is the default Time To Live (TTL) of cache entries in seconds, its default value is 0, which means infinity

  • checkFrequency defines the frequency in seconds for checking and removing expired entries

Add or update entry in the cache

public func setObject<T: Hashable>(_ object: Any, forKey key: T, withTTL: UInt?=nil) {
  • key has to be Hashable
  • if TTL is not specified, cache’s defaultTTL is used for the entry

Retrieve an entry from the cache

  • key has to be Hashable

Retrieve cache statistics

Cache statistics are stored in swift public private(set) var statistics: Statistics Statistics struct contains: - hits the number of cache hits - misses the number of cache misses - numberOfKeys the total number of keys in the cache
